# Sistem Monitoring Budidaya Jamur Tiram Berbasis IoT
# deskripsi
poyek ini merupakan rancangan sistem monitoring jamur tiram berbasis internet of things (IoT) menggunakan ESP32.

# Hardware
-ESP32 DevKit V1
-DHT22
-BH1750
-RTC DS3231
-Relay
-Humadifier
-Exhaust Fan

## Fitur
-monitoring suhu
-monitoring kelembaban
-monitoring umur pertumbuhan
-notifikasi telegram
-Dashboard
